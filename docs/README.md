# FunASR Server 文档

本目录包含 FunASR Server 项目的文档，用于 GitHub Pages 展示。

## 📁 文件说明

- `index.md`: 主页文档
- `_config.yml`: Jekyll 配置文件

## 🚀 本地预览

### 安装 Jekyll

```bash
# 安装 Ruby
sudo apt-get install ruby ruby-dev

# 安装 Jekyll
gem install jekyll bundler
```

### 本地运行

```bash
cd docs
bundle exec jekyll serve
```

访问 `http://localhost:4000/funasr-server/` 查看效果。

## 📝 修改文档

1. 编辑 `index.md` 文件
2. 提交并推送到 GitHub
3. GitHub Actions 会自动部署到 GitHub Pages
